# BA Portfolio - Назар Онищук

Мінімалістичний портфоліо-сайт для Junior Business Analyst, розроблений із HTML/CSS/JavaScript.

## 📋 Структура проекту

```
ba-portfolio/
├── index.html      # Основна сторінка портфоліо
├── styles.css      # Стилізація (мінімалістичний дизайн)
├── script.js       # JavaScript для взаємодії
├── README.md       # Документація
└── .gitignore      # Git конфіг
```

## 🚀 Розгортання на GitHub Pages

### Крок 1: Налаштування репозиторію
1. Перейди в Settings репозиторію
2. Скрол вниз до розділу "GitHub Pages"
3. У полі "Source" вибери гілку **main**
4. Вибери папку **(root)**
5. Клікни **Save**

### Крок 2: Доступ до сайту
Твій портфоліо буде доступний за адресою:
```
https://Nazar000-ba.github.io/-ba-portfolio/
```

> ⏱️ Можливо, знадобиться 1-2 хвилини, щоб GitHub Pages був активований.
